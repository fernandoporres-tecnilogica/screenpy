====================================
Screenpy Docs
====================================
----------------------------------------
a guide to the python screenplay library
----------------------------------------

screenpy is a testing framework that promotes a user-centric, behavioural way of testing that
brings to the forefront what the user is trying to accomplish.  It is primarily designed to be used in
conjunction with Selenium - however, the concepts apply to any behaviour-driven scenario.
